# Sistema-Inventario
Este sistema registra los diferentes movimientos del usuario, puede entrar el administrador que puedes entrar con todos los permisos, usuario consulta solo puede consultar los movimientos y el usuario almacén puede gestionar los productos, categorías y movimientos

## Tecnologías Utilizadas
C# Y SQL SERVER
